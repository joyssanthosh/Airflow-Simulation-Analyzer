
#  Airflow Simulation Analyzer for Smart Interior Layouts

## Project Overview

Airflow Simulation Analyzer is a rule-based decision-support web application designed to evaluate indoor ventilation efficiency based on room layout configurations.

The system simulates airflow movement using a grid-based propagation model, detects dead-air zones, computes ventilation performance metrics, and provides improvement recommendations.

This project focuses on intelligent layout analysis using symbolic (rule-based) AI logic without machine learning implementation.

## Objectives

* Simulate indoor airflow behavior based on vent placement
* Detect low-ventilation (dead-air) zones
* Compute a ventilation efficiency score
* Calculate Airflow Uniformity Index
* Compare multiple layout configurations
* Provide rule-based improvement recommendations
* Visualize airflow using heatmaps and 3D animation

## Core Features

* Layout input module
* Grid-based airflow simulation engine
* Dead-air zone detection
* Weighted ventilation scoring system
* Airflow Uniformity Index calculation
* 2D heatmap visualization

## Technical Approach

The system uses:

* Grid-based airflow propagation logic
* Threshold-based rule classification
* Statistical variance for airflow uniformity
* Weighted scoring formula for ventilation efficiency

No machine learning models are used. The system is implemented as a rule-based AI analytics framework.
